Auth Fusion is a library that allows you to easily add authentication to your Next.js applications. Unlike other libraries, Auth Fusion provides you with the complete code, giving you full control and customization options.

## Features

- Customizable authentication options
  - Choose authentication providers (e.g., credentials, Google, etc.)
  - Additional options for credentials provider (e.g., username, 2FA, etc.)
- Free and open source
- Inspired by Shadcn UI

## Usage

1. Install Auth Fusion as a dependency in your Next.js project.
2. Customize the components and authentication options according to your needs.

## Supported Frameworks

Auth Fusion is built on top of Next.js and can currently be used with Next.js with TypeScript. It will soon be compatible with frameworks like Astro, Remix, and Gatsby.
